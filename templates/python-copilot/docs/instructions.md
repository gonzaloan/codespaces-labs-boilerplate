## Getting Started

1. Open this project in GitHub Codespaces
2. Wait for the environment to set up automatically (Python 3.11 + GitHub Copilot)
3. Add your code inside the `src/` directory
4. Run `pytest` locally to verify your work before pushing
5. Open a Pull Request to trigger the grader

## Tips for using GitHub Copilot

- Use **Tab** to accept Copilot suggestions
- Use **Ctrl+Enter** to see multiple suggestions
- Write a descriptive comment above your function — Copilot uses it as context
- Use **Copilot Chat** (sidebar) to ask questions or request explanations
