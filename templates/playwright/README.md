## Setup

Open in GitHub Codespaces — everything installs automatically.

Or locally:

```bash
npm install
npx playwright install --with-deps chromium
```

## Run your tests

```
npm test
```

## Grading

Push your changes and open a Pull Request against main.
The grader runs automatically and posts a score comment on the PR.
